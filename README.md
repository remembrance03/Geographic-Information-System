# Geographic-Information-System

This repository contains the work of smriti's practical GIS assignment aimed at evaluating core competencies in spatial data handling and GIS analysis using ArcGIS.

The assignment focuses on applying standard GIS workflows to a selected district and municipality of Nepal, integrating spatial and population data, assessing spatial accuracy, and producing meaningful thematic maps.

---

## Objectives

The primary objectives of this assignment are to:

Manage and organize spatial datasets effectively

Perform GIS analysis using ArcGIS tools

Integrate population and attribute data with spatial layers

Assess spatial accuracy and data consistency

Produce clear, meaningful maps following cartographic standards

Document GIS workflows through structured reporting
